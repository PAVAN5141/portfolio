Fashion-MNIST is a dataset of Zalando's article images consisting of a training set of 60,000 examples and a test set of 10,000 examples. Each example is a 28x28 grayscale image, associated with a label from 10 classes. As it was grayscale used single layer channel. First converted 28*28*1 image into 28*28*32 by apllying convolution matrix 5*5 of 32 layers and then applied pooling of 2*2 matrix with stride 2, then again applied convolution of 5*5 matrix which has 64 layers then again pooling with 2*2 matrix, finally arranged all the values into 1D by applying flatten to it, then applied Dropout, Activation function, adam optimizer, to get the output of labels 10.
